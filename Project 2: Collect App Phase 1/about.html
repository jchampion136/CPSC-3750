About Our Page

This is a search application that uses an API from the website Scryfall to fetch real-time Magic: The Gathering information. You can search for cards and view specific cards. Stats about the collection are also available in real-time on the Stats page.

API Link

You can access the Scryfall API directly here: https://scryfall.com/docs/api
